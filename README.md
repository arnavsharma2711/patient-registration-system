## Getting Started

### Prerequisites

- Node.js (v22 or above)
- npm / yarn / bun

### Installation

First, install dependencies

```bash
# Install dependencies
npm install
# or
yarn
# or
bun install
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
